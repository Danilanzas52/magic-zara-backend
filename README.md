# magic-zara-backend
Backend for the zara app (complete this)
REST API with Flask

## Run
``` bash
python3 src/main.py
```
Go to [http://127.0.0.1:5000/?category=coat](http://127.0.0.1:5000/?category=coat), for example

## Requirements
- Flask

## Endpoints
- `/`: Get all stored products
